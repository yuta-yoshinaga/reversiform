・Doxygneフォルダについて 吉永
このフォルダはDoxygenにて作成したソースヘルプドキュメントの保存フォルダです。

★フォルダー説明
・html
	-> Doxygenで作成したソースヘルプドキュメントのHTMLソースの保存フォルダです。

・latex
	-> Doxygenで作成したソースヘルプドキュメントの図の保存フォルダです。

★ファイル説明
・Doxyfile.doxy
	-> DoxygenのGUIフロントエンド用の設定ファイルです。このファイルにDoxygenの各オプションフラグ設定などが保存されています。

・index.html
-> htmlフォルダの中の"index.html"への特殊ショートカットです。相対パス形式のショートカットになっている為、このフォルダをどこに
保存してもダブルクリックでWebブラウザを起動できます。

・XXX.bmp
-> Doxygenプロジェクト用のロゴファイルです。このファイルは無くても問題ありません。

★Doxygenソースヘルプドキュメント作成方法について
※Doxygen対応のコメント、GUIフロントエンドなどが入っている前提での説明です。これらはWebで各自調査してください。

1.Doxyfile.doxyをGUIフロントエンドで読み込む。(私は面倒くさいのでダブルクリックでGUIフロントエンドを起動できるよう拡張子doxyを関連付けました)
2.GUIの"Run"タブを選択する。
3."Run doxygen"を押す。
上記手順でDoxygenドキュメントが作成されます。

※この環境下のDoxyfile.doxyはコピーしてパスが変わっても対応できるように設定ファイル内のパスを設定してあります。各自活用してください。
